About

This website is used to calculate your warm up sets as well as the weights needed on each side. Warmups are calculated as followed, first the bar x 5 x 2, 40% working set x 5, 60% x 3, 80% x 2. It also calculates 1RM(One Rep Max) by using this formula. Working Weight / (1.0278 - (.0278 * 5)), where 5 is the number of reps for your working set. Your 1RM is compared to the Exrx.net weightlifting standards table. There is not a standard for Rows, so the Bench numbers are used instead. Based on warmupreps.com
